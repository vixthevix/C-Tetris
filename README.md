# Tetris in C

Controls:
    Move left - a
    Move right - d
    Rotate clockwise - e
    Rotate anticlockwise - q
    Place down quickly - s

Keeps track of score and returns it upon loss.
To play enter "./tetris {seed}". If no seed provided, randomized.
